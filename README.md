# Smart Research & Publication Management System

A desktop application developed with Qt to help researchers, students and academics organize, track and manage their research lifecycle from idea to publication.

## Project Purpose

The application assists users in:

- Tracking personal research work

- Managing literature and references

- Collaborating with team members

- Conducting internal peer reviews

- Preparing and submitting publications

- Monitoring conferences and submission deadlines

## Main Modules

| Module | Main Features |
| - | - |
| User Module | Registration, login, profile management, user roles (student, researcher, supervisor, admin) |
| Research Module | Research projects, tasks, progress tracking, file attachments |
| Collaboration Module | Team and project member management, shared notes, file sharing, comments |
| Review Module | Internal peer review workflow, feedback collection, review status tracking |
| Publication Module | Manuscript version control, co-author management, target journal/conference tracking |
| Conference Module | Conference database, important dates, submission status, outcome tracking |


## Technology Stack

- Framework: Qt 5 / Qt 6 (Widgets or Qt Quick/QML)

- Language: C++ 

- Database: SQLite (local storage), with potential to extend to PostgreSQL/MySQL

- Build system: CMake or qmake

- Additional Qt modules: Qt SQL, Qt Charts
